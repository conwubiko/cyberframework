CROF REST API

Authentication

All API endpoints (except /api/v1/framework) require a JWT bearer token.

POST /api/v1/token
Request:
{
  "email": "user@example.com",
  "password": "your_password"
}

Response:
{
  "token": "eyJ...",
  "expires": "2026-02-15T12:00:00+00:00"
}

Usage:
Authorization: Bearer eyJ...
Endpoints
MethodEndpointAuthDescription
GET/api/v1/frameworkNoneCROF framework structure (8 functions, 49 controls)
POST/api/v1/tokenNoneObtain JWT token
GET/api/v1/assessmentsJWTList user's assessments
GET/api/v1/assessments/:idJWTFull assessment with scores
GET/api/v1/assessments/:id/scoresSessionAssessment scores (for dashboard charts)
